### 1) initalize uv
    - uv init .
### 2) create virtual environment
    - uv venv
### 3) activate environment
    - source .venv/bin/activate
### 4) install requirement.txt file
    - uv add -r requirement.txt

### 5) The entire project is setup in fastapi, so we can test the project through the postman.
    - python app.py

### 6) setup the langgraph studio (langsmith) , so we can test through it as well
    - langgraph dev
